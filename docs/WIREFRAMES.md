- **Primary**: Deep Blue (#1E40AF)
- **Accent**: Orange (#FB923C) - for progress/achievements
- **Background**: Dark (#0F172A) or Light (#F8FAFC)
- **Text**: Dark text on light bg, light text on dark bg
- **Success**: Green (#22C55E)
- **Warning**: Red (#EF4444)

### Typography
- **Headlines**: Bold, 18-24px
- **Body**: Regular, 14-16px
- **Small**: 12-13px (secondary info)

### Spacing
- Standard padding: 16px
- Card margins: 12px
- Gutter: 8px

### Components
- Rounded corners: 8-12px (cards), 24px (buttons)
- Button height: 44px (mobile, touch-friendly)
- Input height: 40px
- Icons: 24px standard
